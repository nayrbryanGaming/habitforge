# HabitForge Asset Manifest

This manifest documents the visual tokens, design patterns, and asset structure used across the HabitForge ecosystem to ensure long-term design consistency.

## 🎨 Color Palette (Zen Luxury)

| Token | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Blue** | `#2563EB` | Core actions, primary brand identity, active streaks. |
| **Accent Orange** | `#F97316` | Motivation, heat, habit completion celebrations. |
| **Surface White** | `#F8FAFC` | Main background, clean white surfaces. |
| **Slate Dark** | `#0F172A` | Primary typography, dark mode headers, deep contrasts. |
| **Cyan Glow** | `#06B6D4` | Hydration and health habit categorical highlights. |

## 🔠 Typography

- **Headlines**: [Inter](https://fonts.google.com/specimen/Inter) (Bold/ExtraBold) or [Outfit](https://fonts.google.com/specimen/Outfit) for a more modern flair.
- **Body**: [Inter](https://fonts.google.com/specimen/Inter) (Regular/Medium) for high legibility.
- **Micro-copy**: [Outfit](https://fonts.google.com/specimen/Outfit) (SemiBold) with `uppercase` and `letter-spacing: 0.1em`.

## 🖼️ Iconography & Illustrations

- **Icons**: [Lucide React](https://lucide.dev) (Landing Page) & [Material Icons](https://fonts.google.com/icons) (Mobile App).
- **Style**: 2px stroke width, rounded corners (Border Radius: 1.5rem for large containers, 0.75rem for small widgets).
- **Animations**: [LottieFiles](https://lottiefiles.com) — Use the `assets/animations/celebration.json` for all high-value achievements.

## 📁 Asset Directory Structure

- `assets/logo/`: Vector and raster versions of the Hammer & Checkmark logo.
- `assets/images/`: Screenshots of the "Today" dashboard, Analytics, and Premium screens.
- `assets/icons/`: High-resolution PNG icons for Android and iOS splash screens.
- `assets/animations/`: JSON animation files for Lottie integration.
